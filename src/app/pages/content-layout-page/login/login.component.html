  3. உதாரணமாக - ஒரு மாணவரின் EMIS ID 3390XXXX0400018 எனில், அந்த EMIS ID தான் இந்த மதிப்பீட்டிற்கான Login ID. அதில் இறுதியாகவுள்ள 0018-வுடன் “@” symbol சேர்த்து மாணவரின் பிறந்த ஆண்டு 2005 எனில் அந்த மாணவரின் Password 0018@2005 ஆகும்.
  4. மாணவரின் EMIS ID, பிறந்த தேதி இவை அனைத்தும் பள்ளிக் கல்வித் துறை வழங்கிய மாணவரின் திறன் அடையாள அட்டையில் உள்ளது. அதனைப் பயன்படுத்திக் கொள்ளவும்.

  6. மாணவர்கள் மதிப்பீடு மேற்கொள்ளும்போது ஏதேனும் இடையூறினால் மதிப்பீடு தடைபட்டால் மீண்டும் மதிப்பீட்டை முதலிலிருந்து எடுக்க வேண்டும்.
  7. மதிப்பீட்டை முடித்து Complete Assessmentஐ கிளிக் செய்தவுடன் தானாக logout ஆகிவிடும்.
